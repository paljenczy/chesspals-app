# ChessPals Animal Avatar — Image Generation Prompts

## How to use

1. Start with the **Style Reference** prompt below. Generate it and iterate until you like the look.
2. Once you have a style you like, go through all 40 prompts below — paste each one, generate, save.
3. Save each image as `{animal}_{emotion}.png` (e.g., `bee_happy.png`).
4. Run `python3 tools/process_avatars.py` to crop, resize, circle-mask, and place them in the right directories.

## Style Reference (start here)

Generate this first and refine until you're happy with the style:

> A soft watercolor illustration of a cute bee character's face, designed as a circular avatar for a children's chess app. Extreme close-up: the face fills the entire frame, edge-to-edge. Render the artwork inside a perfect circle with a thin solid dark brown outline around the circle edge and a plain white background outside the circle. Round friendly face, big expressive eyes, gentle brushstrokes, muted pastel palette, warm inviting children's storybook style. Small antennae, yellow‑and‑black stripes, calm neutral expression: relaxed round eyes looking forward, flat eyebrows, small closed‑mouth gentle smile, no teeth showing. Clean minimal design. No text, no outer glow, no vignette, and no additional background—only the circular bee face with a thin dark brown border and white outside area.
