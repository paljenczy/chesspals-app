# ChessPals Kid Avatar — Image Generation Prompts

## How to use

1. Use the **same style** as the animal avatars (soft watercolor, children's storybook).
2. Generate all 8 avatars, save each as `kid_avatar_{N}.png` (N = 1–8).
3. Run `python3 tools/process_avatars.py` or a similar script to crop, resize, and circle-mask.

## Base style (copy this prefix for every prompt)

```
A soft watercolor illustration of a cute young {GENDER} character's face, designed as a circular avatar for a children's chess app. Extreme close-up: the face fills the entire frame, edge-to-edge. Render the artwork inside a perfect circle with a thin solid dark brown outline around the circle edge and a plain white background outside the circle.
Round friendly face, big expressive eyes, gentle brushstrokes, muted pastel palette, warm inviting children's storybook style. {FEATURES}, happy cheerful expression: bright smiling eyes, small gentle smile, rosy cheeks. Clean minimal design. No text, no outer glow, no vignette, and no additional background—only the circular child's face with a thin dark brown border and white outside area.
```

## Tips for consistency

- **Generate kid_avatar_1 first** — iterate until the style matches the animal avatars
- **Keep the exact same style words** as the animal prompts — "soft watercolor illustration, children's storybook style, gentle brushstrokes, muted pastel palette"
- If the tool lets you reference a previous image as style input, use one of the animal avatars
- All kids should look ~7–9 years old
- Save as PNG, any resolution (post-processing will resize and circle-crop)
